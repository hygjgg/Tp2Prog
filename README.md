Diferencias arrays vs listas enlazadas

 Arrays

Ventajas	                  

Acceso directo por índice
Fáciles de ordenar	       
Fáciles de recorrer	       

Desventajas

Tamaño fijo, insertar es costoso (mover elementos) y eliminar
requiere correr elementos



 Listas Enlazadas

Ventajas	                   

Insertar y eliminar es O(1)	  
Crecen dinámicamente	        
Mínimo uso de memoria	        

Desventajas

No hay acceso directo (O(n)), no sirven para búsqueda binaria directa
y Recorrer es más lento que array


6. Ventajas y desventajas de usar listas en este TP

 Ventajas

Más flexibles para agregar estudiantes, materias, etc.

No tenés que preocuparte por límites

Insertar ordenado es fácil recorriendo nodos


Desventajas

Implementación más compleja

Búsqueda binaria requiere conversión

Algunos métodos recursivos son más largos
